Plex Setup

Setting up Plex to work with Plexcord is easy. Simply follow the steps below.

Your Plexcord install is running at the following address:

1. Add the Plexcord webhook to Plex.

Unfortunately, Plex have limited the use of webhooks to Plex Pass users only. Plexcord only works with a valid Plex Pass subscription.

Follow Plex's webhook guide to add the URL shown above as a webhook. If you aren't running your Plex server on the same network, this URL may vary and you may need to setup port forwarding on your router.

2. Test Your Setup

Test your setup using the client checking utility. If an ID for a client shows up, all should be good and you are done!

If a client ID doesn't show up, make sure you have followed step 1 correctly and have entered the correct IP address.
